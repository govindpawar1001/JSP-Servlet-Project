#  JSP–Servlet Employee Management System

A **web-based Employee Management System** developed using **JSP, Servlets, JDBC, and MySQL**.  
This application allows an admin to manage employee records efficiently through a clean and user-friendly interface.

---

##  Project Overview

The JSP–Servlet Employee Management System is designed to automate and simplify employee record management in an organization.  
It provides functionality to **add, view, update, delete, and search employees** with secure authentication.

This project demonstrates real-world usage of **Java Web Technologies**, **MVC architecture**, and **database connectivity** using JDBC.

---

##  Project Objectives

- Automate employee record management
- Perform CRUD operations using JSP & Servlets
- Implement MVC architecture
- Integrate MySQL database using JDBC
- Build a secure and scalable web application
- Gain hands-on experience with Java web development

##  Architecture

This project follows the **MVC (Model–View–Controller)** architecture:

- **Model** → Java classes & database logic
- **View** → JSP pages (UI)
- **Controller** → Servlets handling requests & responses
